# 📝 Todo List Application (Django)

A full-stack Todo List web application built using Django. This project allows users to manage their daily tasks efficiently with authentication and CRUD functionality.

---

## 🚀 Features

- 🔐 User Authentication (Signup, Login, Logout)
- ➕ Add new tasks
- ✏️ Edit existing tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 👤 User-specific task management
- 🎨 Responsive UI using Bootstrap
- ⚙️ Admin panel for managing data

---

## 🛠️ Tech Stack

- Backend: Python, Django
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- Deployment: Render (or Railway)
